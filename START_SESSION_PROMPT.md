# Промт для запуска переписывания статей в новой сессии

## Полный промт (скопируй и отправь):

Используй ТЗ из файла rewrite_prompt.md для переписывания всех туристических статей региона Даламан. 

Задача: переписать 82 статьи по существующему списку в progress.md.

Основные правила работы:
1. Новые версии статей сохраняй только в папку content/articles/new/ - НЕ перезаписывая оригиналы!
2. После завершения каждой статьи отмечай её в progress.md как [⏳]
3. Не начинай следующую статью без одобрения предыдущей

Порядок работы:
1. По progress.md найди первую статью со статусом [ ] (не начата)
2. Прочитай оригинальный файл из content/articles/
3. Перепиши её по ТЗ из rewrite_prompt.md
4. Сохрани новую версию в content/articles/new/[имя_файла].md
5. Отметь в progress.md как [⏳] и покажи мне результат
6. Жду, пока я её проверю и скажу "ok" или дам правки
7. Если ок - отмечаю как [✓], я сам копирую из new/ в articles/
8. Переходи к следующей

Начни с первой незавершённой статьи из списка в progress.md.
